# Aplicaciones Python Trainee

Repositorio académico que agrupa los ejercicios y proyectos desarrollados
durante el proceso de formación como Python Trainee.

Cada carpeta corresponde a una aplicación o sistema independiente,
desarrollado con fines educativos, enfocado en la práctica de lógica,
estructuras de datos y manejo básico de archivos en Python.
